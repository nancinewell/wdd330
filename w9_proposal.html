Challenge Proposal

Purpose/Theme

Milton Bradley's game Racko©, to be played locally against the computer. The purpose is to get your 10 randomly dealt cards in order from lowest to highest without reordering them. Instead, by drawing and replacing a card every turn, discarding the replaced card. There are 60 cards in the deck. This game has interesting facts about each number as flavor text on each card which will be generated by an connection to a numbers api.

Audience

This game is geared towards families and friends who enjoy playing games together, so adults and children as young as 3 can play. As long as you understand numbers and where a number falls in comparison to others, you can play.

Data Sources

  • External API: http://numbersapi.com/#42 Retrieve flavor text for numbers.
  • localStorage Set win/loss record to localStorage and retrieve it when page loads and when game ends
